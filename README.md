
# Blazor.Contacts.Wasm

## Descripción
Aplicación de gestión de contactos desarrollada con Blazor WebAssembly.

## Tecnologías Utilizadas
- **Blazor WebAssembly** - Framework web interactivo
- **.NET** - Plataforma de desarrollo
- **C#** - Lenguaje de programación
- **HTML/CSS** - Marcado y estilos

## Requisitos Previos
- .NET SDK 6.0 o superior
- Visual Studio Code o Visual Studio

## Guía de Inicialización

### 1. Clonar el repositorio
```bash
git clone https://github.com/JSebastianBe/Blazor.Contacts.Wasm
cd Blazor.Contacts.Wasm
```

### 2. Restaurar dependencias
```bash
dotnet restore
```

### 3. Ejecutar la aplicación
```bash
dotnet run
```

La aplicación estará disponible en `https://localhost:7108`
